# Esercitazione Data Analytics - Week 4 Day 4 📊
## Ottimizzazione, Data Transformation e Modellazione con Excel e Power Query

### 📌 Panoramica del Progetto
Questa repository contiene i file sviluppati durante l'esercitazione pratica della quarta settimana del Master in Data Analysis, incentrata sull'utilizzo avanzato di Microsoft Excel come strumento di analisi e trasformazione del dato. 

L'obiettivo dell'attività è la gestione di un flusso ETL (Extract, Transform, Load) completo, volto a convertire strutture dati grezze e disorganizzate in un modello analitico solido, preciso e pronto per la reportistica di business.

---

### 🛠️ Tech Stack & Strumenti Utilizzati
* **Microsoft Excel:** Software principale per l'analisi quantitativa e l'elaborazione dei dati.
* **Power Query:** Strumento integrato utilizzato per l'estrazione, la pulizia profonda, la formattazione e la trasformazione automatizzata dei dataset.
* **Funzioni Avanzate e Formattazione:** Applicazione di logiche di calcolo e formattazioni numeriche per garantire l'integrità del dato.

---

### 📊 Obiettivi e Fasi dell'Esercitazione
* **Data Ingestion & Cleaning:** Caricamento dei fogli di lavoro grezzi all'interno dell'editor di Power Query, eliminando record ridondanti, gestendo i valori mancanti e correggendo i tipi di dato.
* **Data Transformation:** Applicazione di passaggi di trasformazione avanzata (es. merge/append di tabelle, normalizzazione dei testi, gestione delle chiavi relazionali) direttamente in Power Query.
* **Data Modeling:** Strutturazione di una base dati relazionale coerente e ottimizzata all'interno della cartella di lavoro per facilitare l'estrazione successiva di KPI.

---

### 📁 Struttura della Repository
* **STRUTTURA ELETTRICA.xlsx:** Cartella di lavoro contenente i dati strutturati e le prime elaborazioni tabellari.
* **MODELLO DATI.xlsx:** File dedicato al modello dati relazionale e alle query di trasformazione finali.
